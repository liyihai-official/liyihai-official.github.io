2. Requirements

If you are using a cluster, make sure you have loaded the right modules. This project requires the modules cmake-3.1.x, OpenMPI-4.x, gcc-12.x and cuda-11.8 at least.

2.1. OpenMPI

There are three general phases of using OpenMPI: installing OpenMPI, building MPI applications, and running MPI applications. The links below take you to the “Quick start” of OpenMPI documentation.

  1. Quick start: Installing Open MPI

  2. Quick start: Building MPI applications

  3. Quick start: Launching MPI applications

2.2. OpenMP

OpenMP does not require an additional installation. Once you add the header file omp.h and the compile flag -fopenmp, the program is ready. You can use the following shell command line:

export OMP_NUM_THREADS=N

to specify the number of threads, or by calling:

#pragma omp parallel num_threads(N)

in the main file. Please see the official OpenMP documentation if you need further information.

2.3. Libtorch

The source files for the C++ API of PyTorch from Meta can be downloaded from the official website. For more information, see the quick start example.
